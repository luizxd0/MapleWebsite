# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024

### Added
- User registration with username, password, and PIN
- User login and logout functionality
- Password change feature for authenticated users
- Account menu with NX credits and MaplePoints display
- Downloads page with installation instructions
- Donation page (placeholder for future implementation)
- Responsive design with Tailwind CSS
- Session-based authentication
- SHA-512 password hashing (compatible with game server)
- Modern React frontend with Vite
- Express.js backend API
- MySQL database integration

### Features
- Monorepo structure for easy development
- Single command to start both frontend and backend
- All pages fit on one screen without scrolling
- Account dropdown menu with user information
- Secure password management
- Game server database compatibility
